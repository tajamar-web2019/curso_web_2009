Formularios

Los formularios se incluyen dentro de la etiqueta form

En ellos se utilizan los siguientes controles

  • input - text
  • text area
  • checkbox
  • radio butons
  • select / options
  • botones

La etiqueta form tiene dos atributos importantes

  • action
  • method
